## Lista de Tareas Dinámica

Este proyecto fue realizado para validar mis conocimientos en desarrollo web. 
Consiste en una lista de tareas dinámica que permite agregar tareas, eliminarlas y marcarlas como completadas.

### Instrucciones
1. Al cargar la página, llenar el formulario con los datos de la tarea.
2. Dar clic en el botón **Guardar** para visualizar la tarea en la tabla.
3. Para eliminar una tarea, dar clic en el icono del bote de basura que aparece al lado de cada registro.

### Tecnologías utilizadas
- HTML5
- CSS
- JavaScript
